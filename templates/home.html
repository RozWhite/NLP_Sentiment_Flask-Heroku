Comment Sentiment Analyzer App


NLP App with Flask

A machine learning end to end flask web app for sentiment analysis with VADER.

VADER (Valence Aware Dictionary for Sentiment Reasoning) is a lexicon and rule-based sentiment analysis tool that detects polarity within our text. This method will give us scores of the following categories: 1. Positive 2. Negative 3. Neutral 4. Compound
